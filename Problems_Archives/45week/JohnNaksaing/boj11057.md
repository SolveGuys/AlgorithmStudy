# 백준 11057 오르막 수
 
### 난이도 : 실버 1
[문제 보러가기](https://www.acmicpc.net/problem/11057)
  
## 동적 계획법
자리수를 쓰는 DP 문제.  
2060KB, 시간 16ms
  
  
## 자료구조
이차원 배열이 필요하다.  
```C++
//dp[i][j] : i번째 j로 시작하는 수
int dp[1001][10];
```
j는 0~9까지이므로 길이 10에 인덱스는 직접 접근이 가능하다.

## 알고리즘
```c++
for (int i = 2; i <= n; i++) 
for (int j = 0; j < 10; j++)
for (int k = 0; k <= j; k++)
{
    dp[i][j] += dp[i - 1][k];
    dp[i][j] %= 10007;
}
```
점화식을 잘 세워줄 필요가 있다.
